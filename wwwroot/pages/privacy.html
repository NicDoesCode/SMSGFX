SMSGFX privacy

One of the design goals for SMSGFX is to create a privacy-first yet fully featured application that runs entirely within the local web browser.

Your projects and configuration all stay within your web browser.

However, we do use a CDN for some external libraries as well as basic web analytics solution, the details of which are below.

Your projects and configuration

While you use the application the following information is created:

  • Your projects.
  • Application settings and preferences.

The above is stored locally within your web browser and stays within your web browser.

Where is SMSGFX hosted?

We're currently hosted on Cloudflare.

Third party libraries

To make the application work the following third party libraries are used:

  • Bootstrap (CSS and JavaScript).
  • Bootstrap Icons (icons).
  • Handlebars (for templating).
  • Moment.js (date and time manipulation).

Currently we link these libraries from external CDNs, which are jsDelivr and Cloudflare.

Analytics

We use Cloudflare Web Analytics, which is a privacy-first analytics platform.

This allows us to collect basic usage metrics and understand general product usage data a little better.
